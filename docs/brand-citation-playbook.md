# Specter Law Brand Citation Playbook

## Canonical Brand Data
- Primary brand name: `Specter Law`
- Secondary brand name: `Specter`
- Common misspelling to target: `Speceter`
- Canonical domain: `https://specterlaw.us`
- Canonical contact page: `https://specterlaw.us/company/contact`
- Primary email: `hello@specterlaw.us`
- Primary X profile: `https://x.com/specterlaw`

## Priority Citation Targets
1. Google Business Profile (if local/legal entity presence is available)
2. LinkedIn Company Page
3. Crunchbase Organization Profile
4. Product Hunt profile
5. Clutch profile
6. G2 profile
7. Capterra profile
8. Wellfound (AngelList) company profile
9. Legal tech directories relevant to US legal software

## Anchor Text Rules For Backlinks
- Use exact brand anchors first: `Specter Law`
- Use partial brand anchors second: `Specter legal AI`
- Use URL anchors third: `specterlaw.us`
- Avoid generic-only anchors (`click here`, `website`) as primary anchors

## Consistency Rules
- Keep exact same brand string everywhere: `Specter Law`
- Do not mix with punctuation variants in profile titles
- Always point profile website field to: `https://specterlaw.us`
- Reuse the same short company description across profiles

## 30-Day Execution Cadence
1. Week 1: publish top-tier profiles (Google, LinkedIn, Crunchbase)
2. Week 2: publish software directories (G2, Capterra, Product Hunt)
3. Week 3: submit to legal-tech and startup directories
4. Week 4: verify indexation and fix inconsistent listings

## Verification Loop
1. Search `site:linkedin.com "Specter Law"`, `site:crunchbase.com "Specter Law"`, etc.
2. Confirm each listing shows the exact canonical domain.
3. Track branded query growth in Google Search Console:
   - `specter law`
   - `specter`
   - `speceter`
